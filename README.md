Лабораторна робота: Робота з Git



Опис

У цій лабораторній роботі ми вивчаємо базові команди Git, роботу з віддаленими репозиторіями GitHub/GitLab, налаштування SSH-ключів та branching.

Мета заняття

Навчитися створювати репозиторії керувати версіями проєкту, працювати з гілками та виконувати злиття merge змін.



Контрольні запитання



1.Що таке атомарний коміт?

&nbsp;  Це коміт, який містить одну логічно завершену зміну.



2.Чим відрізняється GitHub від GitLab?

&nbsp;  GitHub — це найпопулярніша платформа для хостингу коду. GitLab дозволяє розгортати сервіс на власних серверах та має потужні вбудовані інструменти CI/CD.
