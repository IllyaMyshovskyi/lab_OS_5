\# Як зробити внесок Contributing



1\. Зробіть Fork (рощгалуження) цього репозиторію.

2\. Створити гілку (`git checkout -b feature/NewFeature`).

3\. Внести зміни та зробити коміт (`git commit -m 'Add some NewFeature'`).

4\. Відправити зміни (`git push origin feature/NewFeature`).

5\. Створити Pull Request.( Запропонувати зміни автору)
